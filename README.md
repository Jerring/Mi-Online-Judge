Problem_00076：**尺取法**，即反复地推进区间的开头和末尾，求取满足条件的最小区间的方法。主要用于解答求取某个序列满足某种条件的连续的最小区间的问题。可依照以下四个步骤进行：
1. 初始化左右指针
2. 不断移动右指针，直到满足条件 
3. 如果步骤2中的条件无法满足，则终止；否则更新结果  
4. 移动左指针，跳到步骤2
